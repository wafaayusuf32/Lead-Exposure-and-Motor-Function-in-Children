Define variables to test

testing_vars <- c("Age", "Iqf", "Ld72", "Ld73", "Totyrs", "MAXFWT")

for(var in testing_vars) {
  cat("\nVariable:", var, "\n")

  qqnorm(lead[[var]], main = paste("Q-Q Plot for", var))
  qqline(lead[[var]], col = "maroon")

  shapiro <- shapiro.test(lead[[var]])
  cat("Shapiro-Wilk p-value:", shapiro$p.value, "\n")

  hist(lead[[var]],
       main = paste("Histogram of", var),
       xlab = var,
       col = "salmon",
       border = "white")
}

#How would you describe those inferences and what do you observe in terms of the interval width when request higher confidence (i.e. 99% C.I.)? A 90% confidence interval means: “We are 90% confident that the true mean lies within this range.”

A 95% confidence interval means: “We are 95% confident the true mean lies in this range.”

A 99% confidence interval means: “We are 99% confident the true mean is within this wider range.”

#What happens to interval width when confidence increases (e.g. 99% CI)? The higher the confidence level (like 99%), the wider the confidence interval. This is because we need a bigger range to be more sure that the true mean is inside it. A higher confidence level means less precision but more certainty.
